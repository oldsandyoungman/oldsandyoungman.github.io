P14

关于进程在临界区时，是否可以进行处理机调度

  • 如果是操作系统内核程序临界区
    • 不能进行处理机调度和切换，因为会严重影响执行效率
  • 如果是普通资源的临界区（例如打印机），可以调度
    • 这种外界资源本来就很慢，肯定要执行别的东西

进程调度的方式

  • 非剥夺调度方式/非抢占式
    • 只进程能主动放弃
  • 剥夺调度方式/抢占式
    • 时间片用完、优先级更高的进程进入队列

进程的调度和切换区别

  • 进程调度：就绪队列中选择一个进程
  • 进程切换：保存原进程的各种数据 + 恢复新进程的各种数据

912. Sort an Array

  • 快速排序的要点：保证 [start,l) <= val && (r,end]>val
    • 初始条件：l=start+1, r=end
    • 外循环条件：l<=r
    • 内循环条件：l<end && nums[l]<=nums[start] 和 r>start && nums[r]>nums[start]
    • break条件：l>=r
