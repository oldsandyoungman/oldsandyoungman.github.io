48. 旋转图像

给定一个 n × n 的二维矩阵 matrix 表示一个图像。请你将图像顺时针旋转 90 度。

你必须在 原地 旋转图像，这意味着你需要直接修改输入的二维矩阵。请不要 使用另一个矩阵来旋转图像。

提示：

  • matrix.length == n
  • matrix[i].length == n
  • 1 <= n <= 20
  • -1000 <= matrix[i][j] <= 1000
思路

一些线性代数的处理思路，旋转 = 对角线翻转 + 水平翻转

顺时针旋转 = 主对角线翻转 + 水平翻转

逆时针旋转 = 副对角线翻转 + 水平翻转

思路对了应该木有
