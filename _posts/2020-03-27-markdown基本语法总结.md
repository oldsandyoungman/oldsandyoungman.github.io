# 文字效果：
## 超链接：
```	
[域名](www.xxx.com)
```
效果： [百度](https://www.baidu.com)
<br>
## 加粗：
```
**xxx**
```
效果：**Strong text**
<br>
## 斜体：
```
*xxx*
```
效果：*Italic text*
<br>
## 下划线：
```
<u>xxxx</u>
```
效果：<u>Underline text</u>
<br>
## 删除线：
```
~~xxx~~
```
效果：~~Deleted text~~ 
<br>
## 小代码块：
```
` xxx `
```
